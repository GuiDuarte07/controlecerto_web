# Módulo Tickets (Chamados/Ajuda)

- `components/`: UI de lista, detalhe, criação e conversa do chamado.
- `context/`: stores Zustand (usuário e admin).
- `hooks/`: hooks para carregar lista/detalhe e facilitar consumo das stores.
- `schemas/`: validações Zod (criação de chamado e envio de mensagem).
- `services/`: integração com a API (`/api/tickets` e `/api/admin/tickets`).
- `types/`: tipos do módulo e helpers de mapeamento de enums.
